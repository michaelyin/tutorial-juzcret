年产100万套汽车内饰件项目——与东风公司配套，形成开发、应用、改进、再研发的良性循环机制
东风日产、东风汽车公司襄阳基地轻型汽车厂、专用汽车厂和旅行车公司等多家用户的汽车内饰件配套商，大部分都分布在距襄阳200千米以上的地方，而内饰件 又有体积大、不易运输且易对内饰件表面造成磨损等缺点，物流成本高。在宜城投资，可大大减少其物流成本和运输过程中的损坏率，加之宜城良好的投资环境和优 良的产业扶持政策，必将为企业的稳健发展、做大做强奠定坚实的基础。
湖北省宜城市招商局 2016-01-01 2016-12-31
